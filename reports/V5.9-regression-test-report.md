### 2. 权限管理API响应格式问题

**进度**: 3/8测试通过（37.5%）

**失败测试**（5个）：
- ❌ 获取角色详情 - permissions不是数组
- ❌ 删除角色 - 返回500而非200
- ❌ 获取权限列表 - 返回500（新增失败）
- ❌ 创建角色 - 返回500（新增失败）
- ❌ 更新角色 - 返回500

**错误示例**：
```json
// 获取角色详情
{
  "success": true,
  "data": {
    "id": "xxx",
    "permissions": false  // ❌ 应该是数组 []
  }
}
```

**修复建议**：
```typescript
// 确保permissions返回空数组
const roleData = {
  ...role.toJSON(),
  permissions: role.permissions || []
};
```
