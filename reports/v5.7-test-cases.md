## 🧪 功能1：任务依赖关系（P2）

### US-1.1：添加任务依赖关系

#### TC-1.1.01：正常添加依赖关系（FS类型）
**优先级**: P0
**前置条件**:
- 已登录用户
- 存在至少2个未完成的任务（任务A、任务B）

**测试步骤**:
1. 打开任务B详情页
2. 点击"添加依赖"按钮
3. 在任务选择器中选择任务A
4. 选择依赖类型：FS（Finish-to-Start）
5. 点击"确认"保存

**预期结果**:
- ✅ 依赖关系保存成功
- ✅ 任务B详情页显示依赖关系
- ✅ 任务B状态自动变更为"等待依赖"
- ✅ 数据库中创建依赖关系记录

**API测试**:
```bash
# 创建依赖关系
curl -X POST http://localhost:4100/api/v1/dependencies \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <token>" \
  -d '{
    "taskId": "<task_b_id>",
    "dependsOnTaskId": "<task_a_id>",
    "dependencyType": "FS"
  }'
```

**数据库验证**:
```sql
SELECT * FROM task_dependencies 
WHERE task_id = '<task_b_id>' 
AND depends_on_task_id = '<task_a_id>';
```
