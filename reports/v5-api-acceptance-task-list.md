## 1️⃣ 任务查询模块（2个接口）

### 1.1 GET /agent/tasks - 获取我的任务列表

**验收场景**：

#### 正常场景
- [ ] 无参数查询，返回默认分页结果
- [ ] 按状态筛选（todo, in_progress, completed, blocked）
- [ ] 按优先级筛选（low, medium, high, urgent）
- [ ] 分页查询（page, pageSize参数）
- [ ] 返回数据包含正确的任务信息（id, title, status, progress, priority, dueDate）

#### 边界场景
- [ ] pageSize最大值100
- [ ] pageSize最小值1
- [ ] page超过总页数
- [ ] 空任务列表

#### 错误场景
- [ ] 无效的状态参数
- [ ] 无效的优先级参数
- [ ] 未认证（缺少Authorization header）
- [ ] 无效的Token
- [ ] 请求频率超限（429错误）

---
