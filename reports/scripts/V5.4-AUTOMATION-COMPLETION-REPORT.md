## 🚀 执行说明

### 环境要求
- **Node.js**：v14.0.0 或更高版本
- **npm**：v6.0.0 或更高版本
- **Playwright**：v1.40.0 或更高版本

### 安装依赖
```bash
cd team-docs/qa-reports/scripts
npm install -D @playwright/test
npx playwright install
```

### 执行测试

#### 1. 执行所有测试
```bash
npx playwright test
```

#### 2. 执行特定测试文件
```bash
# 登录功能测试
npx playwright test login.spec.js

# 任务列表测试
npx playwright test task-list.spec.js

# 端到端测试
npx playwright test e2e.spec.js
```

#### 3. 执行冒烟测试
```bash
# 方式1：使用脚本（推荐）
chmod +x smoke/run-smoke-tests.sh
./smoke/run-smoke-tests.sh

# 方式2：直接执行Playwright
npx playwright test smoke/login-to-task-list.spec.js
```

#### 4. 在特定浏览器上执行
```bash
# Chrome
npx playwright test --project=chromium

# Firefox
npx playwright test --project=firefox

# Safari
npx playwright test --project=webkit
```

#### 5. 查看测试报告
```bash
npx playwright show-report
```

### 环境变量配置
```bash

export BASE_URL=http://localhost:3101

export TEST_USER=test@test.com
```
